#define closeTo   HC_closeTo

closeTo(aValue, aDelta) - Matches if object is a number close to a given value, within a given delta.

Parameters:
aValueThe double value to compare against as the expected value.
aDeltaThe double maximum delta between the values for which the numbers are considered close.

This matcher invokes -doubleValue on the evaluated object to get its value as a double. The result is compared against aValue to see if the difference is within a positive aDelta.

Example:

  • closeTo(3.0, 0.25)

(In the event of a name clash, don't #define HC_SHORTHAND and use the synonym HC_closeTo instead.)
